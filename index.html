Auto growing textarea

https://github.com/loonkwil/jquery.auto-grow 
$(function() {
  $('textarea').autoGrow();
});

You can set the minimum height of the textarea by setting the height css property or the rows HTML attribute.

You can set a callback function to highlight a string:

$(function() {
  $('textarea').autoGrow({
    highlight: function(str) {
      return str.
        // you have to escape the "&", ">" and the
        // "<" character manualy
        replace(/&/g, '&amp;').
        replace(/>/g, '&gt;').
        replace(/</g, '&lt;').

        replace(
          /(highlight)/g,
          '<mark>$1</mark>'
        );
    }
  });
});

The plugin will recalculate the height of the textarea when you resize the window. If you don't need this (for example you have a fixed width textarea), you can disable. 

$(function() {
  $('textarea').autoGrow({
    recalculateOnResize: false
  });
});
